# Full Stock-and-Flow schema

Includes stocks, flows, inflows, outflows, auxiliary variables, sum
variables, parameters, and all link types.

## Usage

``` r
SchStockFlow
```

## Format

An object of class
[`acsets::BasicSchema`](https://catrgory.github.io/acsets/reference/BasicSchema.html)
(inherits from `S7_object`) of length 1.
