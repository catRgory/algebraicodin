# Log-normal prior

Log-normal prior

## Usage

``` r
prior_lognormal(meanlog = 0, sdlog = 1)
```

## Arguments

- meanlog:

  Mean of the log (default 0)

- sdlog:

  Standard deviation of the log (default 1)

## Value

A prior specification for
[`build_prior()`](https://catrgory.github.io/algebraicodin/reference/build_prior.md)
