# Stock-and-Flow base schema (stocks + sum variables only)

Used as the "foot" type for open stock-flow composition.

## Usage

``` r
SchStockFlow0
```

## Format

An object of class
[`acsets::BasicSchema`](https://catrgory.github.io/acsets/reference/BasicSchema.html)
(inherits from `S7_object`) of length 1.
