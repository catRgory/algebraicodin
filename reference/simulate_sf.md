# Simulate a stock-flow model using deSolve

Simulate a stock-flow model using deSolve

## Usage

``` r
simulate_sf(sf, initial, times, params)
```

## Arguments

- sf:

  A StockFlowModel

- initial:

  Named numeric vector of initial conditions

- times:

  Numeric vector of output times

- params:

  Named list of parameter values

## Value

A data.frame
