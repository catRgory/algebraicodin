# Simulate a discrete ResourceSharer

Simulate a discrete ResourceSharer

## Usage

``` r
simulate_rs_discrete(rs, initial, nsteps, params = list())
```

## Arguments

- rs:

  A discrete ResourceSharer

- initial:

  Named numeric vector of initial conditions

- nsteps:

  Number of time steps

- params:

  Parameter list

## Value

A data.frame with time and state columns
