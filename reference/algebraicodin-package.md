# algebraicodin: Compositional Epidemiological Modeling with 'odin2'

Compositional model building using applied category theory, targeting
the odin2 DSL for high-performance simulation via dust2. Supports Petri
nets, stock-flow diagrams, open systems, and composition via undirected
wiring diagrams. Generates both deterministic (ODE) and stochastic
(discrete-time) odin2 code. Port of AlgebraicPetri.jl and StockFlow.jl
from the AlgebraicJulia ecosystem.
